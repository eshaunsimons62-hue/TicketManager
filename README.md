# 🖥 IT Ticket Manager

A desktop application built in C# and .NET for managing IT support tickets — 
designed to simulate a real-world MSP (Managed Service Provider) environment.

## 📋 Features

- Submit IT support tickets with client name, issue description, and severity level
- View all tickets in a live-updating grid
- Mark tickets as resolved with one click
- All data persisted in a local SQLite database
- Clean, professional UI built with Windows Forms

## 🛠 Tech Stack

- **Language:** C# / .NET 8
- **UI Framework:** Windows Forms
- **Database:** SQLite via Microsoft.Data.Sqlite
- **IDE:** Visual Studio Code

## 💡 Why I Built This

Built to demonstrate core software development skills relevant to MSP environments —
including full project lifecycle experience from design to deployment, database-driven
application development, and client-facing IT support workflows.

## 🚀 How to Run

1. Clone the repo
2. Open terminal in project folder
3. Run `dotnet run`
